# SimpleAHVHostReport

Simple AHV Stats report. Prism Element focus

Merges Nics and LCM Data to provide a specific overview.

Usage:

Extract in the same dir

Run AHV.ps1

Follow the prompts

Open Output.Html in the same dir.
